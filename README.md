# Learning Tracker

A MERN stack application for tracking your learning progress.

## Project Structure

- **client/** - React frontend application
- **server/** - Node.js/Express backend API

## Getting Started

### Frontend (Client)

```bash
cd client
npm install
npm start
```

### Backend (Server)

```bash
cd server
npm install
npm start
```

## Technologies

- **MongoDB** - Database
- **Express.js** - Backend framework
- **React** - Frontend library
- **Node.js** - Runtime environment
